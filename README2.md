# Мой персональный календарь на GitHub Pages

Особенности
- Просмотр календаря по месяцу/году
- Добавление событий на конкретные даты
- Сохранение событий в LocalStorage
- Экспорт/импорт событий в формат JSON

Развертывание
- Вариант A: GitHub Pages (публичный репозиторий)
  1. Создайте репозиторий на GitHub.
  2. Добавьте файлы index.html, styles.css, app.js, README.md.
  3. В настройках репозитория включите GitHub Pages, выберите ветку main (или gh-pages) и папку / (root).
  4. Откройте https://<ваш-логин>.github.io/<имя-репозитория>.

- Вариант B: Netlify/Vercel
  1. Создайте проект из репозитория.
  2. Ваша сборка может быть аналогичной статическому сайту.
